# 🎓 ИИ-Бот «ВЫЖИВШИЙ СТУДЕНТ» (AI-Hub Multimode Bot)

Интерактивный многорежимный Telegram-бот, разработанный на базе фреймворка **Aiogram 3** и интегрированный с локальными и облачными языковыми моделями (**LM Studio / Groq API**). Проект представляет собой развлекательно-обучающую платформу с динамической сменой системных промптов и контекста «на лету».

---

## 🚀 Основной функционал и режимы

Бот поддерживает автоматическое управление историей диалога (до 15 сообщений на пользователя) и переключение между тремя уникальными режимами:

| Режим | Описание механики | Фишка |
| :--- | :--- | :--- |
| **😈 Вредный Профессор** | Интеллектуальный разбор кода на Python и математических задач. Бот сначала остроумно иронизирует над ошибками, а затем выдаёт идеальный академический разбор. | Тонкий юмор, строгая структура, обилие тематических эмодзи. |
| **💀 Попробуй Выживи: Экзамен** | Текстовый квест-симулятор. Профессор задаёт каверзные вопросы (варианты A, B, C, D). | Динамический расчёт **уровня стресса**. При достижении 100% — автоматическое отчисление. |
| **⏳ Учение-Мучение: Вне Времени** | Подменю со списком из 6 великих учёных. Бот интегрирует реальные исторические факты из Wikipedia. | Учёные забавно и философски обыгрывают современный молодёжный сленг. |

## 🛠 Архитектура проекта

Проект построен по модульному принципу, что исключает циклические импорты и упрощает масштабирование:

📂 `my_bot/`  
 ├── 📄 `main_lmstudio.py`  # Главный исполняемый файл (для локального сервера)  
 ├── 📄 `main_groq.py`      # Исполняемый файл для облачного Groq API  
 ├── 📄 `prompts.py`        # Системные промпты (System Prompts) для ИИ и конфигурация ролей  
 ├── 📄 `keyboards.py`      # Генерация клавиатур (ReplyKeyboardBuilder) под каждый режим  
 ├── 📄 `wiki.py`           # Модуль интеграции с API Wikipedia для подгрузки реальных фактов  
 ├── 📄 `.env`              # Файл конфигурации переменных окружения (Токены, Ключи)  
 └── 📄 `README.md`         # Документация проекта

## 💻 Стек технологий

* **Язык программирования:** Python 3.11+
* **Архитектура бота:** Асинхронный `Aiogram 3.x`
* **Работа с LLM:** Локальный сервер `LM Studio` (модель Qwen 3) / `Groq API` (Llama 3.3)
* **Инструменты:** `asyncio`, `requests`, `python-dotenv`.
